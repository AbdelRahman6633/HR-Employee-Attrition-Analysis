# تحليل معدل ترك الموظفين (Employee Attrition Analysis)

مشروع تحليل بيانات HR لفهم أسباب ترك الموظفين وتقليل الـ Turnover.

## الأدوات المستخدمة
- Power BI (للـ Dashboards)
- Power Query وتنظيف البيانات)
- Excel (للتحليل الأولي)

## الإنسايتس الرئيسية
- الموظفين اللي ما أخدوش ترقية من سنتين أو أكتر ليهم احتمالية ترك العمل أعلى بنسبة 37.86%
- الموظفين اللي المسافة من البيت أكتر من 3 ساعات هم اكثر عرضه لترك العمل.
